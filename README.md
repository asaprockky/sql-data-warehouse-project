# SQL Data Warehouse Project

This project demonstrates the design and implementation of a data warehouse using SQL.  
It follows a layered architecture (**Bronze, Silver, Gold**) to transform raw data into clean, structured, and business-ready datasets.

---

## 🏗️ Architecture

- **Bronze** — raw data ingestion from source systems (as-is)
- **Silver** — cleaned and transformed data
- **Gold** — aggregated, analytics-ready data

---

## 🛠️ Technologies Used

- SQL (Microsoft SQL Server)
- T-SQL (views, CTEs, stored procedures)
- Data Warehouse Architecture (Bronze / Silver / Gold)
- ETL concepts (Extract, Transform, Load)

---

## 🚀 Features

- Structured multi-layer data warehouse design  
- Data cleaning and transformation pipelines  
- Analytical-ready datasets for reporting  
- Scalable and maintainable SQL architecture  

---

## 📊 Usage

Run SQL scripts in order:
1. Create database and schemas  
2. Load raw data into Bronze layer  
3. Transform data into Silver layer  
4. Build Gold layer for reporting
